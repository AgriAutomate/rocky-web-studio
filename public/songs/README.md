# Custom Songs Portfolio Audio Files

This directory contains sample tracks for the Custom Songs portfolio section.

## Required Files

- `wheel-of-fortune.mp3` (Liquid DnB - 4:55)
- `loved-out-loud.mp3` (Heavy DnB - 3:02)
- `home-down-under.mp3` (Spoken Word - 1:48)

## File Specifications

- **Format:** MP3
- **Max size:** 10MB per file
- **Bitrate:** 128-320 kbps
- **Sample rate:** 44.1 kHz

## Usage

These files are referenced in `components/services/ServicePortfolio.tsx`.

Audio players are embedded in the portfolio cards.
