## Overview

This case study documents the development and deployment of Rocky Web Studio's AI-powered lead qualification chatbot. The project replaced third-party SaaS chat widgets (Crisp/Drift/Intercom) with a custom solution built using Claude 3 Haiku API, Next.js, and Supabase.

**Key Highlights:**
- **Development Time:** 7 hours (single sprint)
- **Deployment:** 48 hours from start to production
- **Cost:** A$0.40 per 1,000 messages (99.9% savings vs. SaaS)
- **Accessibility:** WCAG 2.1 AA compliant
- **ROI:** Break-even in 7-10 months
