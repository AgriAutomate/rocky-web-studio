# Quick PDF Generation Steps
**File:** `case-study-complete.md`  
**Time:** 5-10 minutes

## Fastest Method: VS Code Extension

### Step 1: Install Extension (One-time)
1. Open VS Code
2. Press `Ctrl+Shift+X` (Extensions)
3. Search: **"Markdown PDF"**
4. Install: **"Markdown PDF" by yzane**

### Step 2: Generate PDF
1. Open: `case-studies/accessibility/case-study-complete.md`
2. Right-click in editor
3. Click: **"Markdown PDF: Export (pdf)"**
4. Wait ~10-30 seconds
5. PDF saved as: `case-study-complete.pdf`

### Step 3: Rename (Optional)
Rename to: `case-study.pdf` for cleaner filename

---

## Alternative: Online Converter

1. Go to: https://www.markdowntopdf.com/
2. Open `case-study-complete.md` in text editor
3. Copy all content (Ctrl+A, Ctrl+C)
4. Paste into converter
5. Click "Convert"
6. Download PDF
7. Save as: `case-studies/accessibility/case-study.pdf`
