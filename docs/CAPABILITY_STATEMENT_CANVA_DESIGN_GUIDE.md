## 1. Brand Colors

### Primary Colors

**Teal Primary: `#218092`**
- **Use for:** Main headings (H1), buttons, call-to-action elements, section dividers, icons
- **Examples:** Page 1 hero title, section headers, download buttons, accent lines
- **Contrast:** Use with white or cream background (4.5:1+ ratio)

**Background Cream: `#fcfcf9`**
- **Use for:** Page backgrounds, card backgrounds, light sections
- **Examples:** All page backgrounds, content card backgrounds, alternate section backgrounds
- **Note:** Slightly off-white for professional, warm appearance

**Text Dark Charcoal: `#1f2121`**
- **Use for:** Body text, subheadings, descriptions
- **Examples:** All paragraph text, bullet points, captions, metadata
- **Contrast:** Excellent on cream background (12.6:1 ratio)

### Accent Colors

**Light Gray: `#a7a9a9`**
- **Use for:** Subtle dividers, secondary text, icons (when needed)
- **Examples:** Footer text, page numbers, subtle borders, disabled states
- **Contrast:** Use sparingly, ensure 4.5:1+ on backgrounds

**Light Teal: `#32b8c6`**
- **Use for:** Hover states, secondary accents, highlights
- **Examples:** Link hover colors, secondary buttons, progress indicators
- **Contrast:** Use with dark text for readability

### Color Usage Rules

**DO:**
- ✅ Use teal (#218092) for all H1 and H2 headings
- ✅ Use dark charcoal (#1f2121) for all body text
- ✅ Use cream (#fcfcf9) for page backgrounds
- ✅ Maintain 4.5:1 contrast ratio minimum

**DON'T:**
- ❌ Use teal on teal (insufficient contrast)
- ❌ Use gray text on cream (may be too light)
- ❌ Mix too many colors (stick to palette)
- ❌ Use gradients or shadows (keep clean)

## 2. Typography Rules

### Heading Hierarchy

**H1 - Cover Page Title**
- **Size:** 48px
- **Weight:** Bold (700)
- **Color:** `#218092` (Teal)
- **Case:** Title Case
- **Alignment:** Center
- **Line Height:** 1.2
- **Example:** "Production-Ready Digital Services for Australian Government"

**H2 - Section Headers**
- **Size:** 28px
- **Weight:** Semibold (600)
- **Color:** `#218092` (Teal) or `#1f2121` (Dark Charcoal)
- **Case:** Title Case
- **Alignment:** Center (cover pages) or Left (body pages)
- **Line Height:** 1.3
- **Spacing:** 60px above, 20px below
- **Example:** "What We Deliver" or "Technical Foundation"

**H3 - Subsections**
- **Size:** 20px
- **Weight:** Semibold (600)
- **Color:** `#1f2121` (Dark Charcoal)
- **Case:** Title Case
- **Alignment:** Left
- **Line Height:** 1.4
- **Spacing:** 30px above, 12px below
- **Example:** "Web Design & Development" or "Security Architecture"

**H4 - Sub-subsections (if needed)**
- **Size:** 16px
- **Weight:** Semibold (600)
- **Color:** `#1f2121` (Dark Charcoal)
- **Case:** Sentence case
- **Alignment:** Left
- **Line Height:** 1.5
- **Example:** "Frontend Framework" or "Backend & Database"

### Body Text

**Paragraph Text**
- **Size:** 14-16px (14px for dense content, 16px for readability)
- **Weight:** Regular (400)
- **Color:** `#1f2121` (Dark Charcoal)
- **Line Height:** 1.5-1.6
- **Alignment:** Left (justified optional, but left is preferred)
- **Spacing:** 12px between paragraphs
- **Max Width:** 600px per column (for readability)

**Bullet Points**
- **Size:** 14-16px (match paragraph text)
- **Weight:** Regular (400)
- **Color:** `#1f2121` (Dark Charcoal)
- **Line Height:** 1.6
- **Spacing:** 8px between bullets
- **Indent:** 20px from left margin
- **Bullet Style:** Simple dot (•) or checkmark (✓) in teal

**Small Text (Footers, Captions)**
- **Size:** 12px
- **Weight:** Regular (400)
- **Color:** `#a7a9a9` (Light Gray) or `#1f2121` (Dark Charcoal)
- **Line Height:** 1.4
- **Use for:** Page numbers, copyright, image captions, metadata

### Font Recommendations

**Primary Font (Headings):**
- **Canva Option:** "Inter Bold" or "Poppins Bold"
- **Alternative:** "Montserrat Bold" or "Roboto Bold"
- **Style:** Modern, clean, professional sans-serif

**Body Font:**
- **Canva Option:** "Inter Regular" or "Open Sans Regular"
- **Alternative:** "Roboto Regular" or "Lato Regular"
- **Style:** Highly readable, web-safe, professional

**Monospace (Code/Technical):**
- **Canva Option:** "Courier New" or "Monaco"
- **Use for:** Code snippets, technical specifications
- **Size:** 12-14px

**Font Rules:**
- ✅ Use maximum 2 font families (one for headings, one for body)
- ✅ Maintain consistent font sizes throughout
- ✅ Use bold/semibold for emphasis, not different fonts
- ❌ Don't mix more than 2 fonts
- ❌ Don't use decorative or script fonts
- ❌ Don't use fonts smaller than 12px

## 4. Design Rules

### Margins & Padding

**Page Margins:**
- **All Pages:** 40px on all sides (top, bottom, left, right)
- **Consistency:** Same margin on every page
- **Canva Setting:** Use "Custom size" and set margins in design

**Content Padding:**
- **Cards/Boxes:** 20px internal padding
- **Text Blocks:** 0px padding (rely on margins)
- **Images:** 10px padding from text

### Section Spacing

**Between Major Sections:**
- **Gap:** 60px between different major sections
- **Example:** Between "What We Deliver" and "Technical Foundation"

**Between Subsections:**
- **Gap:** 30px between subsections within same section
- **Example:** Between "Build Stack" and "Security" on same page

**Heading to Content:**
- **Gap:** 20px between heading and content below
- **Example:** Between H2 and first paragraph

**Paragraph Spacing:**
- **Gap:** 12px between paragraphs
- **Line Height:** 1.5-1.6 for readability

**Bullet Point Spacing:**
- **Gap:** 8px between bullet points
- **Indent:** 20px from left margin

### Icon Guidelines

**Icon Sizes:**
- **Small:** 24x24px (footers, inline with text)
- **Medium:** 32x32px (section headers, feature lists)
- **Large:** 40x40px (hero sections, major features)

**Icon Placement:**
- **Above Text:** 20px gap between icon and text
- **Left of Text:** 12px gap between icon and text
- **Alignment:** Center icons with text baseline

**Icon Style:**
- **Set:** Use Feather icons or Tabler icons (clean, minimal)
- **Color:** Teal (#218092) for primary, Dark Charcoal (#1f2121) for secondary
- **Consistency:** Same icon style throughout document

### Alignment Rules

**Cover Pages (Page 1, Page 8):**
- **Alignment:** Center all content
- **Exception:** Logo (top left), page number (bottom right)

**Body Pages (Pages 2-7):**
- **Alignment:** Left-align all text
- **Exception:** Section headers (center or left, consistent per page)
- **Justification:** Left-align preferred (not justified)

**Grid Alignment:**
- **Columns:** Left-align within columns
- **Rows:** Top-align within rows
- **Consistency:** Same alignment across all grid cells

### Visual Hierarchy

**Priority Order (Most to Least Important):**
1. **H1 Headings** (48px, Bold, Teal)
2. **H2 Section Headers** (28px, Semibold, Teal)
3. **H3 Subsections** (20px, Semibold, Dark)
4. **Body Text** (14-16px, Regular, Dark)
5. **Small Text** (12px, Regular, Gray)

**Visual Weight:**
- **Heavy:** Large, bold, teal color
- **Medium:** Medium size, semibold, dark color
- **Light:** Small, regular, gray color

## 6. Canva-Specific Instructions

### Getting Started

**Step 1: Create New Design**
1. Open Canva
2. Click "Create a design"
3. Select "Custom size"
4. Set dimensions: **8.5" x 11"** (US Letter) or **2550px x 3300px** (for high-res)
5. Name: "Rocky Web Studio Capability Statement"

**Step 2: Set Up Brand Kit**
1. Go to "Brand kit" (Canva Pro) or "Brand" tab
2. Upload logo: Add Rocky Web Studio logo
3. Add colors:
   - Primary: `#218092` (Teal)
   - Background: `#fcfcf9` (Cream)
   - Text: `#1f2121` (Dark Charcoal)
   - Accent Gray: `#a7a9a9`
   - Accent Teal: `#32b8c6`
4. Set fonts:
   - Heading: Inter Bold or Poppins Bold
   - Body: Inter Regular or Open Sans Regular

**Step 3: Use Template as Starting Point**
1. Search: "government report" or "business proposal"
2. Select clean, professional template
3. Remove all template content
4. Keep grid structure if helpful
5. Start with blank page

### Design Workflow

**Recommended Approach:**
1. **Design Page 1 First** (cover page)
   - Get spacing, colors, typography right
   - This becomes your template

2. **Duplicate Page 1** for Pages 2-8
   - Right-click page → "Duplicate"
   - Edit content, keep structure

3. **Create Master Elements**
   - Footer with page number (create once, copy to all pages)
   - Logo element (create once, copy to all pages)
   - Section header style (create once, reuse)

4. **Use Groups**
   - Group related elements (icon + text)
   - Makes copying and positioning easier
   - Right-click → "Group"

### Icon Sources

**Built-in Canva Icons:**
- Search: "Feather icons" or "Tabler icons"
- Style: Clean, minimal, professional
- Size: Adjust to 24px, 32px, or 40px

**Icon Sets to Use:**
- ✅ Feather Icons (built-in)
- ✅ Tabler Icons (built-in)
- ✅ Material Icons (built-in)
- ❌ Avoid: Decorative, 3D, or cartoon icons

**Icon Customization:**
- Change color to teal (#218092) or dark (#1f2121)
- Resize proportionally (hold Shift while resizing)
- Align with text baseline

### Text Formatting

**Creating Text:**
1. Click "Text" → "Add heading" or "Add subheading"
2. Type content
3. Select text → Format toolbar appears
4. Change font, size, color, alignment

**Text Styles to Create:**
1. Format one text element correctly
2. Right-click → "Copy style"
3. Apply to other text elements
4. Or create "Text style" in Brand kit (Canva Pro)

**Text Alignment:**
- Use alignment buttons in toolbar
- Left-align: Body pages
- Center-align: Cover pages, section headers
- Justify: Avoid (left-align preferred)

### Color Application

**Applying Colors:**
1. Select element (text, shape, background)
2. Click color picker
3. Enter hex code: `#218092`
4. Or use brand kit colors (if set up)

**Background Colors:**
1. Select page or element
2. Click "Background" or "Fill"
3. Enter hex code: `#fcfcf9` for cream
4. Or use white (#FFFFFF) for contrast

**Color Contrast Check:**
- Use Canva's accessibility checker (if available)
- Or manually verify: Teal on cream = 4.5:1+ ✅
- Dark text on cream = 12.6:1 ✅

### Spacing & Alignment

**Setting Margins:**
1. Use guides: View → "Show guides"
2. Drag guides to 40px from edges
3. Align content to guides

**Spacing Between Elements:**
1. Select multiple elements
2. Use "Distribute" tool (spacing options)
3. Or manually: Use arrow keys (hold Shift for 10px increments)

**Alignment Tools:**
1. Select multiple elements
2. Use alignment buttons: Left, Center, Right, Top, Middle, Bottom
3. Use "Distribute" for equal spacing

### Grid Creation

**Creating Grids:**
1. Use "Grids" element (Canva Pro) or "Table"
2. Or manually: Use guides and shapes
3. Set columns: 2 or 3 columns
4. Set gutters: 30-40px between columns

**Grid Alignment:**
1. Select all grid elements
2. Use "Align" → "Distribute horizontally"
3. Use "Align" → "Distribute vertically"
4. Ensure equal spacing

### Image Handling

**Adding Images:**
1. Click "Uploads" → "Upload media"
2. Upload logo, screenshots, photos
3. Drag to canvas
4. Resize: Hold Shift, drag corner

**Image Optimization:**
1. Use high-resolution images (300 DPI for print)
2. Compress if file size too large
3. Format: PNG for logos, JPG for photos
4. Check: No pixelation when zoomed

**Image Placement:**
1. Position: Use guides for alignment
2. Size: Maintain aspect ratio (hold Shift)
3. Opacity: Adjust if needed (80-90% for subtle)
4. Borders: Optional 2px teal border

### Common Mistakes to Avoid

**Design Mistakes:**
- ❌ **Too many fonts:** Stick to 2 font families max
- ❌ **Inconsistent spacing:** Use guides and measurements
- ❌ **Low contrast:** Always check 4.5:1 minimum
- ❌ **Busy backgrounds:** Keep clean, use white/cream
- ❌ **Small text:** Minimum 12px, prefer 14-16px
- ❌ **Inconsistent icons:** Use same icon set throughout
- ❌ **Heavy shadows:** Avoid, keep flat design
- ❌ **Gradients:** Avoid, use solid colors
- ❌ **Animations:** Static PDF, no animations

**Content Mistakes:**
- ❌ **Too much text:** Max 60 words per block
- ❌ **Long paragraphs:** Break into shorter paragraphs
- ❌ **Missing page numbers:** Add to every page
- ❌ **Broken URLs:** Verify all links work
- ❌ **Typos:** Proofread multiple times
- ❌ **Inconsistent formatting:** Use text styles

### Troubleshooting

**If Text Looks Too Small:**
- Increase to 14-16px minimum
- Check zoom level (should be 100%)
- Verify export resolution (high quality)

**If Colors Look Wrong:**
- Verify hex codes are correct
- Check color mode (RGB for digital, CMYK for print)
- Test on different screens

**If Spacing Looks Off:**
- Use guides for consistent margins
- Check element alignment (use align tools)
- Verify measurements (40px margins, 20px gaps)

**If Icons Look Inconsistent:**
- Use same icon set (Feather or Tabler)
- Resize to exact sizes (24px, 32px, 40px)
- Use same color (teal or dark)

**If Export Looks Blurry:**
- Export at high resolution (300 DPI)
- Use PDF format (not JPG)
- Check original image quality

## 12. Common Issues & Solutions

### Issue: Text Looks Blurry After Export

**Solution:**
- Export at high resolution (300 DPI)
- Use PDF format (not PNG/JPG)
- Check original text size (minimum 12px)
- Verify zoom level (100% in Canva)

### Issue: Colors Look Different in PDF

**Solution:**
- Verify hex codes are correct
- Check color mode (RGB for digital)
- Test on different screens
- Use brand kit colors (consistent)

### Issue: Spacing Inconsistent

**Solution:**
- Use guides (View → Show guides)
- Set guides to 40px from edges
- Use alignment tools (not manual)
- Check measurements (use ruler)

### Issue: Icons Look Different Sizes

**Solution:**
- Resize to exact sizes (24px, 32px, 40px)
- Hold Shift while resizing (proportional)
- Use same icon set throughout
- Group icon + text for consistency

### Issue: File Size Too Large

**Solution:**
- Compress images before uploading
- Use PDF format (not PNG)
- Reduce image quality slightly
- Remove unnecessary elements

### Issue: Text Overflowing

**Solution:**
- Reduce text size (14-16px max)
- Break into shorter paragraphs
- Use bullet points instead of paragraphs
- Increase text box size
